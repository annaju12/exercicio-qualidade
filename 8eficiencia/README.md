## Problema no código ruim
O uso de índices para percorrer a lista torna o código mais verboso e sujeito a erros.

## Por que o código bom é melhor
O código bom utiliza o for-each, que é mais simples, legível e eficiente.

## O que foi aplicado para melhorar
- Uso de estruturas mais eficientes
- Código mais limpo
- Melhor legibilidade
